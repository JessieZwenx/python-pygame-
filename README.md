# python-game-spacecontrol - 太空资源管理游戏

## 游戏简介
一款基于Pygame开发的太空主题资源管理策略游戏。作为太空站指挥官，你需要管理资源、升级模块、探索未知星域，确保太空站持续运转。

## 系统要求
- Python 3.6+
- Windows 7+/macOS 10.12+/Linux Ubuntu 16.04+
- 至少100MB可用空间

## 安装说明

### 方法一：直接运行（推荐）
1. 确保在 `python-game-spacecontrol - 副本` 目录中
2. 双击 `run.bat` (Windows) 或 运行 `./run.sh` (Linux/Mac)
3. 享受游戏！

### 方法二：从源代码运行
1. 确保已安装 Python 3.6+
2. 进入项目目录：`cd "python-game-spacecontrol - 副本"`
3. 安装依赖：`pip install -r requirements.txt`
4. 运行游戏：`python main.py`

## 游戏操作
- **鼠标点击**: 选择按钮和操作
- **ESC键**: 返回上一界面
- **目标**: 维持太空站运转至少20天
